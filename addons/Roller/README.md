Roller On/Start/Go/Enable/On/Engage
Roller Stop/Off/Quit/End/Disable/Disengage

Roller Roll2 <Name of Roll>		Sets the second roll

Roller TP
Roller Acc
Roller WS
Roller Nuke
Roller Pet
Roller PetNuke
